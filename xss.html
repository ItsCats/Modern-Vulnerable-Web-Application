Cross-Site Scripting (XSS)

Cross-Site Scripting (XSS) is a web security vulnerability that allows an attacker to inject malicious scripts into web pages viewed by other users. This can happen when user-provided data is displayed on a web page without proper sanitization or encoding.

Try These Attacks (Educational Purposes Only)

  • <script>alert('XSS!')</script> (Basic alert)
  • <img src="onerror=alert('XSS')"> (Using an image tag)
  • <a href="javascript:alert('XSS')">Click Me</a> (In an anchor tag)
